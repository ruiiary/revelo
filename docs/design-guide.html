01 디자인 방향
📖
Literary Warmth
종이 위에 글씨를 쓰는 감각. 크림빛 배경, 따뜻한 중성 톤으로 책과 노트의 질감을 디지털로 표현한다.
🎯
Focused & Calm
필사는 집중이 핵심이다. 불필요한 장식 없이 문장과 타이핑에만 시선이 가도록 공간을 비운다.
✨
Rewarding
완료·정답·스트릭 같은 성취 순간에만 감정 색상을 쓴다. 평상시는 차분하게, 달성 시엔 명확하게.
핵심 원칙 — 문장이 주인공이다. UI는 문장을 돋보이게 하는 무대 장치여야 한다. 배경·버튼·아이콘이 문장보다 눈에 띄면 안 된다.

07 모션 & 인터랙션
사용처 Duration Easing 비고
버튼 hover / active 150ms ease opacity 0.75로 살짝 어두워짐
Cover 블록 등장 400ms ease-in-out 단어별 stagger 50ms. 왼→오른 순서
Compare 하이라이트 300ms ease-out 배경색 fade-in. 단어별 stagger 30ms
Progress bar 증가 500ms ease-in-out 단계 전환 시
화면 전환 (step) 250ms ease-out 수평 슬라이드. 이전←, 다음→
바텀 시트 등장 350ms cubic-bezier(0.32,0.72,0,1) 아래에서 위로 슬라이드
완료 이펙트 (Done) 600ms spring 이모지 scale up + bounce
원칙 — 모션은 사용자 행동을 확인해주거나 다음 상태를 예고하는 데만 쓴다. 장식적 애니메이션은 금지. 특히 필사 중 집중을 방해하는 움직임은 넣지 않는다.

핵심 어휘 시스템

Revelo의 언어는 "쌓다" 메타포를 중심으로 통일한다.
책장에 책이 쌓이고, 문장이 층층이 쌓이는 시각적 UI와 언어가 일치해야 한다.

개념 사용할 단어 피할 단어
필사 행위 쌓다, 써보다, 담다 저장하다, 완료하다, 제출하다
필사 단위 문장 콘텐츠, 아이템, 항목
기록 보기 책장, 기록 히스토리, 로그, 데이터
틀린 부분 다시 써볼 문장 오답, 에러, 틀린 것
즐겨찾기 담아두기 즐겨찾기, 북마크, 저장
연속 기록 N일째 이어가고 있어요 스트릭 N일, 연속 달성
